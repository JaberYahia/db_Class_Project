Movie Ranking Project
Overview

The Movie Ranking Project is a web application that allows users to browse, search, and rank movies. Users can view aggregated rankings, rate movies, and see top-rated films. The project combines a React frontend with a Node.js/Express backend and a MySQL database for a full-stack experience.

Features
Browse Movies – View a list of movies with details (title, year, genre, rating).
Search Functionality – Search movies by title or genre.
Rate Movies – Users can submit their own ratings for movies.
Top-Ranked Movies – Display the highest-rated movies based on user votes.
Responsive Design – Works on both desktop and mobile devices.

Technologies Used
Frontend: React.js, HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MySQL
Version Control: Git & GitHub
